# Admin UI for WhatsApp English Lesson Automation

## Features
- Dashboard: user stats, lessons today
- Users: list, filter, update state
- Logs: list, filter

## Tech Stack
- React + Vite
- Tailwind CSS
- Axios

## How to Run Locally
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## API Endpoints
- `GET /api/users`
- `PATCH /api/users/:id`
- `GET /api/logs`

## Folder Structure
- src/pages/
- src/components/
- src/api/
- src/utils/
